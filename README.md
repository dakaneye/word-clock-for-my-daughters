# word-clock-for-my-daughters

Two word clocks, one for each daughter, to be delivered around their 7th birthdays.

- **Emory** — Oct 6, 2030
- **Nora** — Mar 19, 2032

Shared ESP32 + WS2812B platform, per-kid configuration (name, birthday, wood species, lullaby, hidden birthdate acrostic).

See [`docs/superpowers/specs/2026-04-14-daughters-clocks-design.md`](docs/superpowers/specs/2026-04-14-daughters-clocks-design.md) for the full design.

## Lineage

Follows the original word clock for my wife Chelsea, built 2015 and revived 2026, at [`dakaneye/word-clock`](https://github.com/dakaneye/word-clock). Same family, newer generation.
